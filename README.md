## Rodando localmente

Clone o projeto

1. Clone o projeto

```bash
  git clone git@github.com:JonasTiago/jogoforca.git
```

2. Entre no diretório do projeto

```bash
  cd jogoforca
```

3. Instale as dependências

```bash
  npm install
```

4. Inicie o app

```bash
  npm start
```
